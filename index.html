Display and Float Properties: Remember these!

  • Display: Inline
    • appears side-by-side
    • does not accept width or top/bottom margins
  • Display: Block
    • appears vertically, takes up entire line
    • accepts width and top/bottom margin
    • two block elements will combine margins, e.g. If A and B both have 10px margins, the margin between them is only 10px not 20, unlike inline elements.
  • Display: Inline-Block
    • appears side-by-side
    • accepts width and top/bottom margins
  • Floating will cause...
    1. collapsed parent
    2. affects the elements below the float, in this case footer is not clearing
    3. elements become only as wide as their content

Quick Grid System

            box-sizing: border-box;
          }
          .row {
            clear: both;
          }

          footer {
            content: ".";
            display: block;
            clear: both;
            height: 0;
            line-height: 0;
            visibility: hidden;
          }

          [class*="col-"] {
            float: left;
            margin-left: 2%;
          }

          [class*="col-"]:first-child {
            margin-left: 0;
